# 🏋️ ACEest Fitness & Gym — DevOps CI/CD Pipeline

## 📌 Project Overview
This project implements a complete **DevOps CI/CD pipeline** for the ACEest Fitness & Gym application.  
The system is built using a Flask-based web application and demonstrates modern DevOps practices including automation, testing, containerization, and deployment.

---

## 🚀 Features
- Client management (add/view clients)
- Fitness program tracking
- Progress monitoring
- Calorie calculation system
- Web-based UI interface

---

## 🛠️ Tech Stack

| Category | Tools Used |
|--------|-----------|
| Backend | Python, Flask |
| Testing | Pytest |
| CI/CD | Jenkins |
| Code Quality | SonarQube |
| Containerization | Docker |
| Container Registry | Docker Hub |
| Orchestration | Kubernetes (Minikube) |
| Cloud Deployment | AWS EC2 |
